# Track 2 – NVT Roadmap: Java + K8s + SigNoz

Practical study material for a **reference corporate environment**: Java (Spring Boot) on Kubernetes (OKE), Config Server, GitHub Actions pipelines, and observability with SigNoz and OpenTelemetry.

## Audience

SREs or developers operating Java applications on Kubernetes who need to understand deploy order, health checks, troubleshooting, and observability in this stack.

## Suggested order

1. **Reading** – `nvt-roadmap/00_overview_environment.md` and `nvt-roadmap/01_stack.md`.
2. **Reference** – `nvt-roadmap/02_sre_checklist_java_k8s.md` (use during incidents and deploys).
3. **Labs** – Complete `nvt-roadmap/labs/` in order (lab-01 through lab-05).
4. **Practice** – `nvt-roadmap/exercises/exercises-nvt-roadmap.md`.

## Contents

- `nvt-roadmap/` – Overview, stack description, SRE checklist, labs, and exercises for the Java+K8s+Config Server+SigNoz environment.
