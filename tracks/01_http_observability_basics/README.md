# Track 1 – HTTP & Observability Basics

Study material for the **SRE Study Guide** fundamentals: HTTP requests, APIs, load balancing, observability (metrics, logs, traces), APM with SigNoz, and SRE best practices.

## Audience

Anyone learning how requests flow through systems, how to observe them, and how to apply SRE practices. No prior SRE experience required.

## Suggested order

1. **Chapters** – Read `chapters/01_http_requests.md` through `chapters/06_sre_best_practices.md` in order.
2. **Exercises** – Do the exercises in `exercises/` after each corresponding chapter (or in batch).
3. **Labs** – Follow `labs/01.md` and the example application / workshop-instrumentation labs for hands-on OpenTelemetry and SigNoz.
4. **Reference** – Use `detailed_log_analysis.md` for deeper log analysis concepts.

## Contents

- `chapters/` – Theory (HTTP, REST/gRPC/GraphQL, load balancing, observability, APM SigNoz, SRE best practices).
- `exercises/` – Practice exercises per chapter.
- `labs/` – Hands-on labs (Node.js app with OTel, workshop instrumentation).
- `detailed_log_analysis.md` – Reference material on log analysis.
